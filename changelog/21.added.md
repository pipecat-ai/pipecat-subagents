- Added `PgmqBus`, a distributed agent bus backed by PGMQ (PostgreSQL Message Queue). Each instance broadcasts to peer queues sharing the channel prefix, providing fan-out semantics on top of PGMQ's point-to-point queues. Install with `pip install pipecat-ai-subagents[pgmq]`. A runnable example is included under `examples/distributed/pgmq-handoff/`.
