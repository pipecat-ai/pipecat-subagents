- Added `LLMContextAgent`, a subclass of `LLMAgent` that owns an `LLMContext` and an `LLMContextAggregatorPair`. Its pipeline is built as `[user_aggregator, llm, assistant_aggregator]`, so subclasses no longer need to wire the context plumbing themselves. The constructor accepts optional `context`, `user_params`, and `assistant_params`, and the aggregators are exposed via the `user_aggregator` / `assistant_aggregator` / `context` properties (available from `on_ready` onwards).
